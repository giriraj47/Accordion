What is HTML?
HTML (HyperText Markup Language) is the standard language used to structure and present content on the web. It defines elements like headings, paragraphs, links, and images.
What is CSS used for?
CSS (Cascading Style Sheets) controls the design and layout of web pages. It allows developers to change colors, fonts, spacing, and positioning of HTML elements.
What is JavaScript?
JavaScript is a scripting language that adds interactivity and dynamic behavior to websites. It enables functions like animations, form validation, and user interaction.
What is the difference between class and id in HTML?
A class can be used multiple times across different elements, while an id is unique and used to identify a single element on the page.
What is responsive web design?
Responsive web design ensures that a website looks and works well on all devices, from desktops to smartphones, by adjusting layout and elements automatically.
